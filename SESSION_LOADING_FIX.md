# 会话加载问题已修复

## 问题
- 移除了自动加载所有会话的代码
- 导致点击项目后历史会话不加载，Agent 列表空白

## 修复方案

### 1. 启动时智能加载
- Chat 项目：自动加载会话（因为默认选中）
- 普通项目：
  - 未折叠的项目：自动加载
  - 折叠的项目：不加载

### 2. 点击展开时加载
- 用户点击展开项目时
- 自动触发会话加载
- 显示历史会话列表

## 行为说明

### 启动时
```
Chat 项目 ✅ 加载会话
项目A (展开) ✅ 加载会话
项目B (折叠) ⏸️ 不加载
项目C (折叠) ⏸️ 不加载
```

### 用户操作
```
点击展开项目B → ✅ 自动加载会话
点击展开项目C → ✅ 自动加载会话
```

## 优势

### 性能提升
- ✅ 启动时只加载必要的会话
- ✅ 减少 IPC 调用
- ✅ 降低内存占用

### 用户体验
- ✅ Chat 项目立即可用
- ✅ 展开项目即时加载
- ✅ 加载状态清晰可见
